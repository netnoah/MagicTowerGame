# 魔塔游戏项目记忆

## 门设计规则（已验证）

### 三大铁律

1. **死锁避免**：玩家在遇到必须要过的门之前，必须能拿到相应的钥匙
   - 钥匙位置必须在门的"解锁侧"
   - 从起点出发，检查可达区域内是否有足够钥匙

2. **封闭区域有门**：任何封闭区域都必须有门作为入口
   - 被墙包围的区域必须有门才能进入
   - 封闭区域内的物品玩家必须能获取

3. **门旁无绕行**：门旁边的地板必须改成墙
   - 门所在的行/列，门必须是唯一的地板格
   - 玩家无法从门的上下/左右绕过门

### 验证方法

设计地图后，用BFS/DFS从起点模拟玩家路径：
1. 找出所有可达区域（不经过门）
2. 检查可达区域内是否有足够钥匙
3. 模拟开门后，检查新可达区域
4. 重复直到所有区域可达

### 地图1修复案例

**问题**：
- 门(12,13)把地图分两半，钥匙在门后面，死锁
- 封闭区域(左上/右上)没有入口
- 门旁边有地板可绕行

**解决方案**：
- 把钥匙移到玩家初始可达区域(10,15)
- 在封闭区域墙上开地板+门：(4,2)和(20,3)
- 门旁边地板改成墙：行3列4和行2列20改成墙

- 门类型：yellow, blue, red, green

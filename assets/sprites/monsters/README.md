## 资源规范

### 文件命名

动画帧命名格式: `<动画名>_<帧号>.png`

```
monsters/slime_green/
├── idle_00.png    # 待机动画第0帧
├── idle_01.png    # 待机动画第1帧
├── idle_02.png    # ...
└── ...
```

### 图片要求

| 项目 | 规格 |
|-----|------|
| 格式 | PNG (透明背景) |
| 尺寸 | 任意 (自动缩放到 1.5x tile = 48x48 范围内) |
| 方向 | 面向右边 (自动水平翻转生成左边方向) |
| Pivot | 底部中心 (自动计算) |

3. 没有资源时会显示红色占位符

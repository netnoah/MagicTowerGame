# 魔塔游戏项目 - Claude 指令

## 项目概述

这是一个使用 Python + Pygame 开发的经典魔塔RPG游戏。

## 地图设计规则（必须遵守）

### 门设计三大铁律

1. **死锁避免**：玩家在遇到必须要过的门之前，必须能拿到相应的钥匙
2. **封闭区域有门**：任何封闭区域都必须有门作为入口
3. **门旁无绕行**：门旁边的地板必须改成墙，确保门是唯一通道

### 地图设计验证清单

每次设计/修改地图时，必须检查：

```
- [ ] 钥匙可达性：从起点出发，每遇到一个门，之前是否有对应钥匙？
- [ ] 封闭区域：所有封闭区域是否有门入口？
- [ ] 门旁绕行：门旁边是否都是墙（不能有地板可绕行）？
- [ ] 门数量平衡：钥匙数量是否足够开所有门？
- [ ] 门有意义：不开门能否到达门的另一侧？（能=门无意义）
```

### 正确的门设计示例

```
封闭房间入口:
  墙墙墙墙墙墙墙
  墩 物品  墩
  墙墙墙门墙墙墙    ← 门堵住唯一入口，旁边是墙
  墙      墙

通道门:
  墙墙墙墙墙墙墙
  墙      墙
  墙墙墙门墙墙墙    ← 通道1格宽，门堵住唯一通道
  墙      墙
  墙墙墙墙墙墙墙
```

### 错误的门设计示例

```
可绕行的门（错误）:
  墙墙墙墙墙墙墙
  墩      墩
  墩  门  墩        ← 门旁边是地板，可以从旁边绕过！
  墩      墩
  墙墙墙墙墙墙墙

死锁设计（错误）:
  起点 → 墙墙墙门墙墙墙 → 钥匙
         ↑
      没钥匙开不了门，拿不到钥匙，死锁！
```

## 编码规范

- **不可变原则**：创建新对象，不修改原对象
- **测试覆盖率**：目标 80%+
- **文件大小**：控制在 800 行以内
- **函数大小**：控制在 50 行以内
